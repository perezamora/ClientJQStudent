# WEB APP MVC (JQUERY) - Student

Demo de un cliente basado en ASP.NET MVC 5 (JQUERY)

## Introducción

Se realiza un CRUD en MVC pero utilizando solo comunicacion AJAX + JQUERY en donde se realiza un simple gestion de estudiantes.

Las funcionalidades principales se basan:
* Create (Insertar alumno)
* Read (Leer alumno)
* Update (Actualizar alumno)
* Delete (Borrado alumno)
* GetAll (Listado de los alumnos)

El cliente se comunica con una web api via protocolo HTTP, en formato JSON. En el servidor WEB API se persisten los datos en SQL SERVER utilizando
Entity Framework.

Tener en cuenta activar el cross domain dentro de la webapi, para poder realizar comunicaciones entre diferentes servidores.
